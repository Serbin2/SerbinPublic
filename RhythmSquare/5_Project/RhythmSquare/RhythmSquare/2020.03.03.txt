
03.03(화)

게임 흐름

1. 게임 외적, 게임에 필요한 여러가지 기능
게임 루프 내에서 수정이 불가능한 정보들, 라이브러리 내부에 포함되며
싱글턴으로 구성되어 전역성을 가진다.

-> time, key, mouse, sound... 등등

2. 게임 내적, 게임에 필요한 여러가지 기능
외부 기능에서 필요한 정보를 다시 이벤트, 명령 등으로 변경하여 게임 프로세스에
전달한다. 게임 프로세스와 소통이 가능하며, 게임 프로세스 내에서 수정이 가능하다.
프로젝트 내부에 포함되며, 싱글턴으로 구성되어 전역성을 가진다.

-> ui, ui input, graphics, game fream, game sound ...

3. 메인 프로세스
메인 프로세스에서는 실제 게임 진행에 필요한 루프를 돌린다.
메인 프로세스 내에서 종료를 하면 게임이 종료된다.

sceen, game, score, event

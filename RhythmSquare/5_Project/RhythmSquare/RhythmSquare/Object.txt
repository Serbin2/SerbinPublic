 
 오브젝트 설계는 비슷한 기능을 하는 오브젝트 끼리 레이어로 나눈다. 
 레이어 별 성능은 다음과 같습니다.

 0 Layer
 Back Ground 배경, Scene의 기본 레이어. -1 레이어들은 SceneManager에서 관리하며 render한다.
 보유 속성 -> 그리기

 1 Layer
 버튼, UI (Menu, Pop) 등을 소유한 레이어 사용자의 입력대로 처리되며 마우스 커서와 충돌체크 등을 한다.
 충돌이 된 이후에는 특정 이벤트를 발생시키며 플레이어에게 전달된다.
 보유 속성 -> 충돌, 그리기

 2 Layer
 InGame 상에 등장하는 오브젝트들 인게임 버튼, 유닛 등 게임 판정에 사용된다. 아직 어케할지 생각 중.
 게임의 판정은 충돌하거나 하는 것이 아닌, 

 3 Layer
 InGame 상에 등장하는 오브젝트들, 고민인건 애니메이션을 하는 오브젝트들 (유닛, 몬스터)을 따로 모아놓을까 생각 중.

 4 Layer
 무조건 이펙트
